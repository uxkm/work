### 필수 단축키

빠른 파일 열기 : `ctrl + p`
명령어 보기 : `ctrl + shift + p`
자동정렬
- `ctrl + a` 전체선택 후
- `ctrl + k + f`

---

### 에디터 단축키

왼쪽 사이드바 on/off: `ctrl + b`
하단 패널 on/off: `ctrl + j`
Explorer바 열림(포커스): `ctrl + shift + e`
에디터 분리: `ctrl + \`
모든 코드 접기: `ctrl + k + 0`
모든 코드 펼치기: `ctrl + k + j`
영역 코드 접기: `ctrl + shift + [`
영역 코드 펼치기: `ctrl + shift + ]`
주석 on/off: `ctrl + /`

---

### 검색 단축키

파일명검색: `ctrl + e`
전체검색: `ctrl + shift + f`
파일 내 전체 검색: `ctrl + f`
다음 선택 찾기: `ctrl + d`
프로젝트 검색: `ctrl + r`

---

### 디버그 단축키

디버그: `f5`
디버그 중지: `shift + f5`
브레이크 포인트 on/off: `f9`
