Collapse


html문서 내 호출 및 기본 구조

	
	<!-- 기본 -->
	<div data-modules-ollapse>
		<div class="sticky-inner">
			마크업 내용...
		</div>
	</div>
	<a href="javascript:;" data-modules-collapse class="accordion-trigger" aria-expanded="false">내용보기 클릭!</a>
	<div class="accordion-panel" role="region">
		내용이 들어갑니다!!!
	</div>

	<!-- 커스텀 옵션 사용 -->
	<div class="acc_parent">
		<a href="javascript:;" data-modules-collapse="parent:.acc_parent" class="accordion-trigger" aria-expanded="false">내용보기 클릭!</a>
		<div class="accordion-panel" role="region">
			내용이 들어갑니다!!!
		</div>

		<a href="javascript:;" data-modules-collapse="parent:.acc_parent" class="accordion-trigger" aria-expanded="false">내용보기 클릭!</a>
		<div class="accordion-panel" role="region">
			내용이 들어갑니다!!!
		</div>
	</div>

script에서 호출

	
	// 기본 옵션없이 호출
	$(Element).collapse();


	// 커스텀 옵션 사용
	$(Element).collapse({
		parent:".acc_parent",
	});

최소한의 CSS

	
.accordion-panel{
	display: none;
}
.accordion-panel.selected{
	display: block;
}

options

Name Type default description
parent selector null 그룹핑 할 부모요소
targetClass selector accordion-panel collapse 컨텐츠 클래스 이름
animate boolean false 애니메이션 부여 여부(css transition 사용)
animationClass string collapsani 애니메이션 중 부여될 클래스 이름

Method

  • 없음

Event

  • 없음, show, hide 구현 예정
