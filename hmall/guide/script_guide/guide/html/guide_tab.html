Tab


html문서 내 호출 및 기본 구조

	
	<!-- 기본 -->
	
	<ul class="ui-tab" role="tablist">
		<li role="presentation" class="ui-active"><a href="#home" aria-controls="home" role="tab" data-modules-tab>Tab1</a></li>
		<li role="presentation"><a href="#profile" aria-controls="profile" role="tab" data-modules-tab>Tab2</a></li>
		<li role="presentation"><a href="#messages" aria-controls="messages" role="tab" data-modules-tab>Tab3</a></li>
		<li role="presentation"><a href="#settings" aria-controls="settings" role="tab" data-modules-tab>Tab4</a></li>
	</ul>
	
	<!-- Tab panes -->
	<div class="tab-content">
		<div role="tabpanel" class="tab-pane ui-active" id="home">tabContent1</div>
		<div role="tabpanel" class="tab-pane" id="profile">tabContent2</div>
		<div role="tabpanel" class="tab-pane" id="messages">tabContent3</div>
		<div role="tabpanel" class="tab-pane" id="settings">tabContent4</div>
	</div>

script에서 호출

	
	// 기본
	$(Element).tab();

최소한의 CSS

	
.tab-content > .tab-pane.ui-active{
	display: block;
}
.tab-content > .tab-pane {
	display: none;
}

options

- 없음

Method

Name description
$(element).tab('show') 해당하는 텝 컨텐츠 활성화

Event

Name description
show.hui.tab show instance 메소드가 호출 될 때 즉시 발생
shown.hui.tab 노출 처리 css 전환이 완료 된 후 발생
hide.hui.tab hide 메소드가 호출시 발생
hidden.hui.tab 숨김처리 css 전환이 완료된 후 발생
