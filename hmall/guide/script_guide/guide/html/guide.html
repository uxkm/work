Intro

웹 인터페이스 개발을 위한 jQuery 기반 UI Framework입니다. 보편적인 웹 개발에서 필요한 UI 모듈 대부분을 포함하고 있습니다.
현 페이지에서는 설치 및 모듈 초기화 가이드를 제공합니다.

2-1. Selector - 태그에 삽입하여 호출

ui스크립트 모듈에 사용되는 셀렉터는 모두 data-modules-[모듈이름]을 사용합니다.

	
	<!-- 모듈 사용 예:기본 옵션 없이 호출) -->
	<div data-modules-sticky>
		마크업 내용...
	</div>

	<!-- 모듈 사용 예:커스텀 옵션 사용) -->
	<div data-modules-sticky="padding:0;relativeTarget:#wrap;">
		마크업 내용...
	</div>

2-2. Selector - <script>에서 직접 호출

스크립트로 직접 호출하는경우 data-modules-[모듈이름]에서 "모듈이름" 부분이 함수명이 됩니다.

	
	// 기본 옵션없이 호출
	$('SelectorName').sticky();


	// 커스텀 옵션 사용
	$('SelectorName').sticky({
		padding:0,
		relativeTarget : '#wrap'
	});
