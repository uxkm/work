Dropdown


html문서 내 호출 및 기본 구조

	
	<!-- 기본 -->
	<div class="ui-dropdown">
		<a id="drop1" href="#" data-modules-dropdown aria-expanded="false"><span>드롭다운 클릭!</span></a>
		<ul class="dropdown-menu" role="menu" aria-labelledby="drop1">
			<li><a href="#">내용1</a></li>
			<li><a href="#">내용2</a></li>
		</ul>
	</div>

script에서 호출

	
	// 기본 호출
	$(Element).dropdown();

최소한의 CSS

	
.ui-dropdown{
	position: relative;
}
.ui-dropdown .dropdown-menu{
	display: none;
}
.ui-dropdown.ui-active .dropdown-menu{
	display: block;
}

options

- 없음

Method

Name description
$(element).dropdown('toggle') 드롭다운박스 토글

Event

Name description
show.hui.dropdown show instance 메소드가 호출 될 때 즉시 발생
shown.hui.dropdown 노출 처리 css 전환이 완료 된 후 발생
hide.hui.dropdown hide 메소드가 호출시 발생
hidden.hui.dropdown 숨김처리 css 전환이 완료된 후 발생
