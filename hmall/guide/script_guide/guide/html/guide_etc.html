앱 스와이프

앱에서의 스와이프 동작과 웹에서의 스와이프 동작이 충돌날 경우 사용하는 메서드로 앱 담당 개발자님께 공유 받았습니다.
* setPagingDrag 메서드는 메인페이지에서만 존재 한다고 합니다.(메서드 사용 시 해당 이름의 메서드가 존재 하는지 필수로 체크 후 사용 바랍니다.) 


window.AndroidJS.setPagingDrag("true")  // 앱에서 스와이프동작 우선 처리
window.AndroidJS.setPagingDrag("false")  // 웹에서 스와이프동작 우선 처리


window.appYn;   // 앱 여부 "Y"/"N"
window.appOpsyNm;   // 앱 타입 "android"/"ios"
