Window popup


html문서 내 호출 및 기본 구조

	
	<!-- 기본 -->
	<!-- Button trigger modal -->
	<button type="button" class="ui-test-btn" data-modules-winpopup="href:guide_pop_test.html;width:800;height:600;">
		버튼을 이용한 팝업 호출
	</button>
	<a href="guide_pop_test.html" class="ui-test-btn"  data-modules-winpopup="width:800;height:600;">
		a링크를 이용한 팝업 호출
	</a>

script에서 호출

	
	UI.util.openPopup('guide_pop_test.html', {
		'width' : 800,
		'height': 600
	})

options

Name Type default description
name string 팝업 고유 이름 팝업 고유 이름을 부여합니다. 팝업위에 팝업이 뜰 경우 동일한 이름을 사용하면 현재 팝업에서 링크가 이동됩니다.
href string a링크 href 값 팝업 호출을 a링크가 아닌 다른 테그를 이용하는 경우 옵션 href 값으로 전달
width Number 600 팝업 Width
height Number 400 팝업 height
resizable string no yes || no IE만 해당됨
scrollbars string no yes || no IE만 해당됨
scrollbars string no yes || no IE만 해당됨
