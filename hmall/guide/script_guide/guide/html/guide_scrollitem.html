scrollItems


html문서 내 호출 및 기본 구조

	
	<!-- 기본 -->
	<div data-modules-scrollItems>
		<span class="ui-nav">1</span>
		<span class="ui-nav">2</span>
		<span class="ui-nav">3</span>
		<span class="ui-nav">4</span>
		<span class="ui-nav">5</span>
		<span class="ui-nav">6</span>
		<span class="ui-nav">7</span>
	</div>

	<!-- 커스텀 옵션 -->
	<div data-modules-scrollItems="align:center;">
		<span class="ui-nav">1</span>
		<span class="ui-nav">2</span>
		<span class="ui-nav">3</span>
		<span class="ui-nav">4</span>
		<span class="ui-nav">5</span>
		<span class="ui-nav">6</span>
		<span class="ui-nav">7</span>
	</div>

script에서 호출

	
	// 기본 옵션없이 호출
	$(Element).scrollItems();


	// 커스텀 옵션 사용
	$(Element).scrollItems({
		'align':'center'
	});

최소한의 CSS

	
[data-modules-scrollItems]{width: 400px;overflow-x: auto;white-space: nowrap;position: relative;font-size: 0;}
[data-modules-scrollItems].btn-type{width: 450px;overflow: hidden;}
[data-modules-scrollItems].btn-type .ui-nav{width: 50px;}
.ui-nav{display: inline-block;width: 100px;height: 30px;border: 1px solid #000;font-size: 12px;}
.ui-nav.ui-active{background:#f00;}

options

Name Type default description
activeClass string ui-active 아이템 활성화 클래스 명
align int or string 0 숫자일 경우 클릭된 요소를 좌측 기준으로 n번째 정렬, 문자일 경우 "center" 사용 클릭된 요소 가운데 정렬

Method

Name description
$(element).scrollItems('initActiveItem'); 활성화 클래스가 붙은 요소를 찾아 정렬 시켜준다.
$(element).scrollItems().goActive(n) n번째 요소 활성화 및 정렬

Event

Name description
init.scrollItems scrollItems 모듈 초기화시 발생
active.scrollItems 아이템 선택 시 발생
scrollEnd.scrollItems 스크롤 액션 후 발생
