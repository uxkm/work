script에서 호출

	
	// 기본 옵션없이 호출
	$(Element).moreview();


	// 커스텀 옵션 사용
	$(Element).moreview({
		"height":42
	});

최소한의 CSS

	
[data-modules-moreview].ui-hidden .ui-inset{height: 60px;overflow: hidden;}
[data-modules-moreview].ui-active .ui-inset{height: auto;}

options

Name Type default description
height int 60 컨텐츠가 이 높이를 초과 하면 스크립트가 적용됩니다.
openText String 더보기 Open 텍스트
closeText String 닫기 Close 텍스트
hiddenClass String ui-hidden hidden이 적용될 클래스 명
