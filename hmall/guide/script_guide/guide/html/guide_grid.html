cascadingGrid


html문서 내 호출 및 기본 구조

	
	<!-- 기본 -->
	<div data-modules-cascadingGrid>
		<div data-items></div>
		<div data-items></div>
		<div data-items></div>
		<div data-items></div>
	</div>

	<!-- 커스텀 옵션 사용 -->
	<div data-modules-cascadingGrid="padding:5;">
		<div data-items></div>
		<div data-items></div>
		<div data-items></div>
		<div data-items></div>
	</div>

script에서 호출

	
	// 기본 옵션없이 호출
	$(Element).cascadingGrid();


	// 커스텀 옵션 사용
	$(Element).cascadingGrid({
		padding:0
	});

최소한의 CSS

	
[data-modules-cascadingGrid]{
	margin: 0 auto;
	position: relative;

}
[data-items]{
	position: absolute;
	top:0;
	left:0;
	width: 150px;
	background: #333;
	color: #fff;
}

options

Name Type default description
padding Number 0 그리스 요소 사이 간격

Method

Name description
$(element).cascadingGrid('mount') 그리드 새로고침, 비동기로 그리드 리스트 추가 시 호출

Event

  • 없음
